 🚀 Features

Real-time Chat: Integrated a chat system where messages appear instantly with dynamic visual feedback.
Video Calling: Developed web-based calling functionality with ringtone play/stop features using HTML5 Audio API.
 Interactive UI: Features include mute/camera toggle, active speaker highlighting, and simulated live call interactions.

🛠️ Tech Stack

Languages: HTML5, CSS3, JavaScript.

APIs: HTML5 Audio API for call management.
